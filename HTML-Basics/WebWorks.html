The Internet

What's the internet?

The internet is an interconnection of computer networks which allow them to communicate and share resources and information.

How the internet works

Servers are special computers directly connect to the internet.
Webpages are files on the server's hard drive.
Everything connected directly or indirectly to the internet has a special address called IP (Internet protocol) address.
IP address help computers find each other.
Our day-to-day computers are called clients because the are indirectly connected to the internet through an ISP (Internet Service Provider ).
When information travels across the internet, computers break them down into smaller pieces called packets.
When the packets reach their destination the reassemble themselves in the correct order.
Anywhere where two or more parts of the internet intersect there is a router.
Routers direct packets around the internet.

How the internet works in less than 5 mins

What is Domain Name?

A domain name is the unique, human-friendly address used to access a website on the internet, such as "google.com".

What DNS is and how it works?

DNS (Domain Name System) is the internet's "phone book" that translates human-readable domain names (like www.example.com) into IP addresses (like 192.0.2.1)

What are browsers and how they work?

A browser is an application that allows users to access and view websites on the internet by retrieving files from a webserver and rendering them as a readable webpage.
It works using a client-server model: the browser acts as the "client" to request information from a web server, which then sends the website's content, like HTML files, back to the browser to be displayed. Examples include Chrome,Safari etc.

What is a Markup language?

A markup language is a system that uses tags or codes to annotate and define the structure and presentation of digital documents, such as web pages.
Unlike programming languages, it is not executed but is read and rendered by computers to format text, control layout, and provide instructions on how content should be displayed.
A key example is HTML, the standard language for creating web pages, which uses tags to tell a browser how to display words, images, and other elements.

What is Frontend Development?

Front-end development is the practice of creating the user-facing parts of websites and applications, focusing on everything a user sees and interacts with in their browser.
It involves using for example HTML for structure, CSS for styling, and JavaScript for interactivity to build the layout, design, and functionality of a site.
This role ensures a visually appealing, intuitive, and responsive user experience across different devices.

What is http?

HTTP stands for Hypertext Transfer Protocol, a fundamental set of rules that web browsers and servers use to communicate over the internet.
It operates on a request-response basis, where your browser sends an HTTP request to a server, and the server sends back an HTTP response containing the requested data.
This protocol enables the transfer of files like HTML documents, images, and more, which is how you view web pages.
Though HTTPS is preferred:
HTTP vs. HTTPS: While http:// can be used for many transfers, https:// encrypts the connection, making it a safer choice for sensitive information like credit card details.

What is the WWW and how it works?

The World Wide Web (WWW) is a system of interconnected, publicly accessible webpages that run on the Internet.
It works by using web browsers to request information from web servers, which are computers that store websites.
Key technologies like HTTP, HTML, and URLs allow the browser to retrieve files, translate the code into a readable webpage, and display it to the user.
